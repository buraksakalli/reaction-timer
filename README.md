## Installation

Use package manager [yarn](https://yarnpkg.com/en/docs/install) or npm to install node_modules.

```bash
yarn install
or
npm install
```

## Usage

```bash
react-native run-android #for running on Android
react-native run-ios #for running on iOS
```
